# Pharmacovigilance Literature-Based ICSR Case Processing Portfolio

## About the Project

This project demonstrates the end-to-end processing of literature-based Individual Case Safety Reports (ICSRs) using published case reports. The portfolio follows standard pharmacovigilance practices, including case validation, MedDRA coding, seriousness assessment, expectedness evaluation, WHO-UMC causality assessment, medical review, and regulatory narrative writing.

This project was developed to strengthen practical knowledge of pharmacovigilance case processing and drug safety reporting.

---

## Project Objectives

- Perform literature-based ICSR case processing.
- Apply MedDRA coding for adverse events.
- Assess case validity, seriousness, and expectedness.
- Perform WHO-UMC & Naranjo Scale causality assessment.
- Develop regulatory-style case narratives.
- Demonstrate practical pharmacovigilance workflow.
